# RhymesAI Project

A rhyme generation app with a FastAPI backend and minimal frontend, using the `devyatilov/trained_rugpt3_rhyme_large` model.

## Prerequisites
- Windows 10 or later (or any OS supporting Python 3.12)
- Python 3.12
- 16GB+ RAM recommended (model is resource-intensive)
- Node.js (optional, for serving frontend locally)
